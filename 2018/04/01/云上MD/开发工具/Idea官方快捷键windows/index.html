IntelliJ IDEA官方 Mac 和 Windows 及 Linux 快捷键 中文版

IntelliJ Idea
  • 快捷键

以下快捷键，皆译自官方原文档。

记住这些快捷键

Mac 描述 Windows/Linux
⌃ ⇧ Space 智能代码完成 Ctrl + Shift + Space
Double ⇧ 到处搜索 Double Shift
⌘ ↩︎ 显示意向操作和快速修复 Alt + Enter
⌘ N, ⌃ ↩︎ 生成代码 Alt + Ins
⌘ P 参数信息 Ctrl + P
⌥ ⌘ ↑ 扩展选择 Ctrl + W
⌥ ⌘ ↓ 收缩选择 Ctrl + Shift + W
⌘ E 最近的文件弹出 Ctrl + E
⇧ F6 改名 Shift + F6

通用

Mac 描述 Windows/Linux
⌘0 … ⌘9 打开相应的工具窗口 Alt + #[0-9]
⌘ S 保存全部 Ctrl + S
⌘ ⌥ Y 同步 Ctrl + Alt + Y
⌘ ⇧ F12 切换最大化编辑器 Ctrl + Shift + F12
⌥ ⇧ I 用当前配置文件检查当前文件 Alt + Shift + I
⌃ §, ⌃ ` 快速切换当前方案 Ctrl + BackQuote (`)
⌘ , 打开设置对话框 Ctrl + Alt + S
⌘ ; 打开项目结构对话框 Ctrl + Alt + Shift + S
⌘ ⇧ A 找到行动 Ctrl + Shift + A

试调

Mac 描述 Windows/Linux
F8 / F7 跨过/步入 F8 / F7
⇧ F7 / ⇧ F8 智能进入/退出 Shift + F7 / Shift +F8
⌥ F9 运行到游标 Alt + F9
⌥ F8 评估表达式 Alt + F8
⌘ ⌥ R 恢复程序 F9
⌘ F8 切换断点 Ctrl + F8
⌘ ⇧ F8 查看断点 Ctrl + Shift + F8

搜索/替换

Mac 描述 Windows/Linux
Double ⇧ 到处搜索 Double Shift
⌘ F 查找 Ctrl + F
⌘ G / ⌘ ⇧ G 查找下一个/上一个 F3 / Shift + F3
⌘ R 替换 Ctrl + R
⌘ ⇧ F 在路径中查找 Ctrl + Shift + F
⌘ ⇧ R 在路径中替换 Ctrl + Shift + R
⌃ G 选择下一个发生 Alt + J
⌃ ⌘ G 选择所有发生 Ctrl + Alt + Shift + J
⌃ G 取消选择发生 Alt + Shift + J

编辑

Mac 描述 Windows/Linux
⌃ Space 基本代码完成 Ctrl + Space
⌃ ⌘ Space 智能代码完成 Ctrl + Shift + Space
⌘ ⇧ ↩︎ 完成陈述 Ctrl + Shift + Enter
⌘ P 参数信息（在方法调用参数中） Ctrl + P
⌃ J , 快速查找文档 Ctrl + Q
⇧ F1 外部文件 Shift + F1
⌘ + mouse 简要信息 Ctrl + mouse
⌘ F1 在插入符号处显示错误的描述 Ctrl + F1
⌘ N, ⌃ ↩︎ 生成代码… Alt + Insert
⌃ O 覆盖方法 Ctrl + O
⌃ I 实现方法 Ctrl + I
⌘ ⌥ T 用…包围 Ctrl + Alt + T
⌘ / 注释/取消注释行列 Ctrl + /
⌘ ⌥ / 注释/取消注释模块 Ctrl + Shift + /
⌥ ↑ 扩展选择 Ctrl + W
⌥ ↓ 收缩选择 Ctrl + Shift + W
⌃ ⇧ Q 上下文信息 Alt + Q
⌥ ↩︎ 显示意向操作和快速修复 Alt + Enter
⌘ ⌥ L 重新格式化代码 Ctrl + Alt + L
⌃ ⌥ O 优化导入类库 Ctrl + Alt + O
⌃ ⌥ I 自动缩进行（s） Ctrl + Alt + I
⇥ / ⇧ ⇥ 缩进/取消缩进选定行 Tab / Shift + Tab
⌘ X 将当前行剪切到剪贴板 Ctrl + X , Shift + Delete
⌘ C 将当前行复制到剪贴板 Ctrl + C , Ctrl + Insert
⌘ V 从剪贴板粘贴 Ctrl + V , Shift + Insert
⌘ ⇧ V 从近期缓冲区中粘贴… Ctrl + Shift + V
⌘ D 重复当前行 Ctrl + D
⌘ ⌫ 在插入符号处删除行 Ctrl + Y
⌃ ⇧ J 智能合并行 Ctrl + Shift + J
⌘ ↩︎ 智能分割行 Ctrl + Enter
⇧ ↩︎ 开始新的一行 Shift + Enter
⌘ ⇧ U 在插入符号或选定的块上切换单词大小写 Ctrl + Shift + U
⌘ ⇧ ] / ⌘ ⇧ [ 选择直到代码块结束/开始 Ctrl + Shift + ] / [
⌥ ⌦ 删除到单词结尾 Ctrl + Delete
⌥ ⌫ 删除到单词开始 Ctrl + Backspace
⌘ + / ⌘ - 展开/折叠代码块 Ctrl + NumPad+ / -
⌘ ⇧ + 展开全部 Ctrl + Shift + NumPad+
⌘ ⇧ - 收缩全部 Ctrl + Shift + NumPad-
⌘ W 关闭活动编辑器标签 Ctrl + F4

重构

Mac 描述 Windows/Linux
F5 复制 F5
F6 移动 F6
⌘ ⌫ 安全删除 Alt + Delete
⇧ F6 重命名 Shift + F6
⌃ T 重构当前 Ctrl + Alt + Shift + T
⌘ F6 更改签名 Ctrl + F6
⌘ ⌥ N 一致 Ctrl + Alt + N
⌘ ⌥ M 提取方法 Ctrl + Alt + M
⌘ ⌥ V 提取变量 Ctrl + Alt + V
⌘ ⌥ F 提取字段 Ctrl + Alt + F
⌘ ⌥ C 提取常量 Ctrl + Alt + C
⌘ ⌥ P 提取参数 Ctrl + Alt + P

导航

Mac 描述 Windows/Linux
⌘ O 转到类 Ctrl + N
⌘ ⇧ O 转到文件 Ctrl + Shift + N
⌘ ⌥ O 转到符号 Ctrl + Alt + Shift + N
⌃ ← / ⌃ → 转到下一个/上一个编辑器选项卡 Alt + Right/Left
F12 返回到上一个工具窗口 F12
⎋ 转到编辑器（从工具窗口） Esc
⇧ ⎋ 隐藏活动或上一个活动窗口 Shift + Esc
⌘ L 转到行 Ctrl + G
⌘ E 最近的文件弹出 Ctrl + E
⌘ ⌥ ← / ⌘ ⌥ → 导航后退/前进 Ctrl + Alt + Left/Right
⌘ ⇧ ⌫ 导航到上一个编辑位置 Ctrl + Shift + Backspace
⌥ F1 在任何视图中选择当前文件或符号 Alt + F1
⌘ B, ⌘ Click 转到声明 Ctrl + B , Ctrl + Click
⌥ Space / ⌘ Y 打开快速定义查找 Ctrl + Shift + I
⌘ ⌥ B 转到类型声明 Ctrl + Shift + B
⌘ U 转到超级方法/超级类 Ctrl + U
⌃ ↑ / ⌃ ↓ 转到上一个/下一个方法 Alt + Up/Down
⌘ ] / ⌘ [ 移至代码块结束/开始 Ctrl + ]/[
⌘ F12 文件结构弹出 Ctrl + F12
⌃ H 类型层次 Ctrl + H
⌘ ⇧ H 方法层次 Ctrl + Shift + H
⌃ ⌥ H 调用层次 Ctrl + Alt + H
F2 / ⇧F2 下一个/上一个突出的错误 F2 / Shift + F2
F4 / ⌘ ↓ 编辑源/查看源文件 F4 / Ctrl + Enter
⌥ Home 显示导航栏 Alt + Home
F3 切换书签 F11
⌥ F3 用助记符切换书签 Ctrl + F11
⌃0 … ⌃9 转到编号书签 Ctrl + #[0-9]
⌘ F3 显示书签 Shift + F11

编译和运行

Mac 描述 Windows/Linux
⌘ F9 编译项目 Ctrl + F9
⌘ ⇧ F9 编译选定的文件，包或模块 Ctrl + Shift + F9
⌃ ⌥ R/D 选择配置并运行/调试 Alt + Shift + F10/F9
⌃ R/D 运行/调试 Shift + F10 / F9
⌃ ⇧ R / ⌃ ⇧ D 从编辑器运行上下文配置 Ctrl + Shift + F10

用法搜索

Mac 描述 Windows/Linux
⌥ F7 / ⌘ F7 查找用法/查找文件中的用法 Alt + F7 / Ctrl + F7
⌘ ⇧ F7 突出显示文件中的用法 Ctrl + Shift + F7
⌘ ⌥ F7 显示用法 Ctrl + Alt + F7

VCS /本地历史记录

Mac 描述 Windows/Linux
⌘ K 将项目提交给VCS Ctrl + K
⌘ T 从VCS更新项目 Ctrl + T
⌘ ⇧ K 推送提交 Ctrl + Shift + K
⌃ V ‘VCS’快速弹出 Alt + BackQuote (`)

实时模板

Mac 描述 Windows/Linux
⌘ ⌥ J 用实时模板来环绕 Ctrl + Alt + J
⌘ J 插入实时模板 Ctrl + J
